Get balances

This API endpoint retrieves the account balances, including available and locked funds.

Request

HTTP Method & Endpoint

GET /api/v1/user/balances

HTTP headers Header

KeyValueDescription
Acceptapplication/jsonSpecifies that the response format should be JSON.

Query parameters Query

KeyTypeRequiredSignDescription
client_keystringYesYesThe API access key.
signaturestringYesNoSigned value.

Request example

curl -X GET \
  https://example.com/api/v1/user/balances?client_key=01h6tn69wfcpy5q5x3vpb3x9me&signature=ba5df26991273c746960ce5238c6479e8ca6116381ac46cea96ffd30fafed082 \
  -H "Accept: application/json"

Response

Response parameters

KeyTypeDescription
client_keystringThe API access key.
available_fundsstringThe amount of funds available.
locked_fundsstringThe amount of funds currently locked.
currencystringThe currency code (e.g., BRL, VND).

Response example

{
  "client_key": "01h6tn69wfcpy5q5x3vpb3x9me",
  "available_funds": "16055.90",
  "locked_funds": "0.00",
  "currency": "BRL"
}
